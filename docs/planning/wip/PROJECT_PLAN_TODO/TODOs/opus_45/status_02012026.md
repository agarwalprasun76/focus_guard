## 🗺️ ROADMAP: NEXT PHASE OPTIONS

### Option 1: Complete Override Integration ✅ DONE
**Effort: 2-3 days | Impact: High** - *Completed Feb 1, 2026*

Classification is now fully connected to the override/budget system:

1. **Content-type aware budgets** ✅
   - EDUCATION:EDUCATIONAL: 60 min/day (no penalty)
   - EDUCATION:ENRICHMENT: 30 min/day
   - ENTERTAINMENT:NEUTRAL: 15 min/day
   - ENTERTAINMENT:DISTRACTION: 10 min/day (screenshot + parent notify)
   - GAMING:DISTRACTION: 5 min/day (screenshot + parent notify)

2. **Override penalties by content type** ✅
   - EDUCATION: 0s penalty per extra override
   - ENTERTAINMENT:DISTRACTION: 120s penalty per extra override
   - GAMING:DISTRACTION: 180s penalty per extra override

3. **Blocked page shows classification** ✅
   - `/api/should_block` now returns `classification` and `budget_status`
   - `BlockingDecision.to_dict()` includes all info for blocked page display
   - Budget status includes: time_used_formatted, time_budget_formatted, percentage_used

### Option 2: News & Streaming Classifiers
**Effort: 3-5 days | Impact: Medium**

Complete the domain-specific classifier coverage:

1. **News sites** (`news_rules.py`)
   - Distinguish article topics (tech news vs celebrity gossip)
   - Major news domains: CNN, BBC, NYT, etc.
   - Classify by URL path (/politics, /entertainment, /tech)

2. **Streaming sites** (`streaming.py`)
   - Netflix, Hulu, Disney+, HBO Max, Prime Video
   - Twitch (gaming streams vs educational)
   - Almost always ENTERTAINMENT/DISTRACTION

### Option 3: Rule Evolution System
**Effort: 1-2 weeks | Impact: High (long-term)**

Build feedback loop for smarter rules:

1. **Classification logging**
   - Log every classification with full context
   - Track user overrides ("user said productive, we said distraction")

2. **Weekly analysis job**
   - LLM reviews logs and suggests rule updates
   - "Users frequently override for 'tutorial' in title → add to education keywords"

3. **Rule versioning**
   - Track rule changes over time
   - Rollback capability

### Option 4: Visual/Screen Classification (Future)
**Effort: 2-4 weeks | Impact: Very High**

True "agent watching screen":

1. Periodic screen capture
2. Vision LLM analysis
3. Works for any app (not just browser)
4. Privacy-first: local processing option

**Challenges:** Privacy, cost, latency
