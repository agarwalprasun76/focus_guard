FocusGuard Windows Installer Scripts

- auto_build_and_copy.bat: Rebuilds the native host and copies it to the correct location.
- install_focusguard_extension.bat: Installs the extension files and native host, and sets up registry/manifest for Chrome.
- inno_setup_script.iss: Use Inno Setup to create a user-friendly installer for end users.

For manual testing, run the batch scripts as your normal user (not as Administrator).
